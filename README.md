# Story-Architect

Story-Architect is an AI-powered creative partner designed to guide writers from initial concept through structural outlining to detailed prose expansion. 

Built with **Flet** (Flutter for Python), it uses a "Critic-in-the-Loop" architecture to ensure narrative consistency and high-quality prose through iterative self-evaluation.

## Key Features
- **Phase I: Ideation** - Generate divergent concept branches.
- **Phase II: Outlining** - Interactive beat cards with "pinned" story details.
- **Phase III: Expansion** - AI-driven prose generation with an integrated "Literary Critic" for iterative refinement.
- **Persistence** - Automatic client-side saving and full project exports via JSON.

## Tech Stack
- **Frontend:** [Flet](https://flet.dev/)
- **AI Orchestration:** OpenAI-compatible API
- **State Management:** Pydantic + Flet Client Storage
